Basic Snippet

Want to search the web? Try Google!

This is what the html code for this look like

  <p>
  Want to search the web?
  Try <a href="https://google.com" id="the-best-link" class="banana"  target="_blank">Google </a>
  </p>

What does each element in this snippet do?

  • p: this creates a "paragraph" element in which everything else is contained. Just like a paragraph in a book, this seperates blocks of text from each other.
    • W3 Schools Paragraph Tag
    • Mozilla on the Paragraph Tag
  • a: this tag is know as the anchor element and is what tells the web page that you are linking to another webpage, and its oppening and closing tags surround the display text for the link, in this case "Google"
    • W3 Schools Hyperlink Tag
    • Mozilla Anchor tag information
  • href: this attribute signals the speficic web page that you are linking to
  • id: this attribute identifies the element "a" to your css styles sheet, and an id can only be used once per html page, so it is the most specific identifier you can use
  • class: this attribute is similar to an id, but can be used for many element, and is therefore a less specific identifier. If an element has a class and an id, the id will overrule the styles set by the class.
  • target: this attribute tells your browser where to open the link you've created, which in this case is a new page.
